Spark: API Proof of Concept Katie Roberts Spark is an iPhone/iPad application for browsing creative inspiration on the go. Users can view different categories of projects including UI/UX, branding, graphic design and web design and development. Spark uses data from the Behance API, displaying images, names, and author data.
